# SideVue

A VueJs powered Sidebar to drop into your projects with ease.

> It needs vueify to work.

## Installation

```
npm install --save sidevue
```

## Usage

```
Vue.component('sidevue', require('sidevue'));
```

You can control your sidebars with the vue event system.
```
    'sidevue-open'
    'sidevue-close'
    'sidevue-toggle'
```
